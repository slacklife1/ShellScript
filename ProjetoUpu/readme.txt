INFORMAÇÕES:


makefile.sh		<--- Script de Instalção do Universal Packing Unpacking Linux
upu.sh			<--- Script do upu
makefile.sh -R		<--- Desinstala o upu


DESCRIÇÃO DE USO:
	
	Após a instalação chamar o comando upu de qualquer lugar

	Para o uso da Compactação de arquivo

		upu [ARQUIVO]

	Para a Descompactação de Arquivo (.zip, .rar, .tar, .bz2, .tar.bz2)

		upu [ARQUIVO.ALGO]


INSTALAÇÃO:

	./makefile.sh

DESINSTALAÇÃO
	
	./makefile.sh -r
